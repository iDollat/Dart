      How to Play Dart Score Tracker

      1. Choose a game mode (301 or 501)
      2. Add at least 2 players
      3. Each player starts with the selected score (301 or 501) and aims to reduce it to exactly 0
      4. Players take turns throwing 3 darts per round
      5. Click on the dartboard buttons to record scores
      6. If a player's score would go below 0 in a round, the entire round is skipped
      7. The first player to reach exactly 0 wins!

      Keyboard Shortcuts:

      • Numbers 1-9: Score single numbers
      • D + number: Score doubles (e.g., D2 for double 2)
      • T + number: Score triples (e.g., T3 for triple 3)
      • M: Miss (0 points)
      • Ctrl+Z: Cofnij ostatni rzut
