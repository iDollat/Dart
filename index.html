    How to Play Dart Score Tracker

    Game Setup

    • 🎯 Choose a game mode (301 or 501)
    • 👥 Add at least 2 players
    • 🎲 Each player starts with the selected score

    Game Rules

    • 🎯 Players take turns throwing 3 darts per round
    • 📊 Click on the dartboard buttons to record scores
    • ⚠️ If a player's score would go below 0, the round is skipped
    • 🏆 First player to reach exactly 0 wins!

    Keyboard Shortcuts

    • 1-9 Score single numbers
    • D + number Score doubles (e.g., D2 for double 2)
    • T + number Score triples (e.g., T3 for triple 3)
    • M Miss (0 points)
    • Ctrl+Z Undo last throw

    Features

    • ↩️ Undo throw: Use the "Undo" button or Ctrl+Z
    • 💡 Checkout hints: See suggestions for finishing the game
    • 📱 Compact view: Toggle for a more compact button layout
    • 📊 Score drawer: View all players' scores and round history
